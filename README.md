## Running Tests
```
go/      cd go && go test ./...
jvm-langs cd jvm-langs && ./gradlew test
rust     cd rust && cargo test
python   cd python && pytest
```

## Learning Approach

- **Rust**: Concepts first, algorithms as consequence
- **C++**: Multiple implementations per problem (different approaches, same output)
- **Other languages**: Follow the roadmap sequentially
